## Usage
clone the project
```bash
git clone https://github.com/leogaudin/fract-ol.git
```
compile the program
```bash
make && make clean
```
and execute the program
```bash
./fractol [fractol_name]
```

## Note
- MinilibX is the graphical library used for rendering.
